##  **Key Features**

- **Beautiful modern GUI** ‚Äî manage economy without typing commands
- **Player balance management** ‚Äî view, give, take, set exact amounts
- **Mass operations** ‚Äî give/take/set money to/from multiple players at once
- **Fast player search** ‚Äî built-in search bar + filters (online/offline/selected)
- **Global economy statistics** ‚Äî total balance, average, richest/poorest players
- **Transaction history** ‚Äî view detailed logs of all operations
- **Quick actions** ‚Äî fast give/take/set for individual players
- **Multi-language support** ‚Äî English & Russian included, easy to add more
- **PlaceholderAPI ready** ‚Äî perfect for custom scoreboards & holograms
- **Fully Vault-compatible** ‚Äî works with any Vault economy plugin

##  **Commands & Permissions**

| Command                     | Description                              | Permission                   |
|-----------------------------|------------------------------------------|------------------------------|
| `/economygui gui`           | Open the main Economy GUI                | `economygui.gui`             |
| `/economygui reload`        | Reload config and messages               | `economygui.reload`          |
| `/economygui reset`         | Reset search and filters                 | `economygui.reset`           |

**Permissions:**

- **`economygui.admin`**  
  Grants **full access** to all commands, features and update notifications  
  *(Default: OP)*

- **`economygui.gui`**  
  Allows opening the EconomyGUI interface

- **`economygui.reload`**  
  Allows reloading the configuration and messages

- **`economygui.reset`**  
  Allows resetting search and filters in the GUI

- **`economygui.mass-give`**  
  Allows mass giving money to selected players

- **`economygui.mass-take`**  
  Allows mass taking money from selected players

- **`economygui.give`**  
  Allows giving money to individual players
